4. 定位板设计

    - 标准定位板采用压铆螺柱链接做支撑，不方便小量设计
    - 图中直接采用非标螺柱进行支撑，使用m2沉头螺丝和m2螺母进行固定
    - 非标螺柱可3d打印, 不过最好使用cnc
